Important FAQ

what is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web document. It represents the page so that programs can change the document structure, style, and content.

How do you select on element from the DOM?

We can select an element from the DOM in many ways using Methods, like Document.GetElementById(), Document.GetElementsByTagName(), Document.GetElementsByClassName(), document.querySelector(), or Document.querySelectorAll() atc.

What is event delegation in the context of the DOM. and why is it useful?

Event delegation is a design pattern in JavaScript used to efficiently manage and handle events on multiple child elements by attaching a single event listener to a common ancestor element.
and it useful because it helps us create dynamic and interactive website by modifying the content of web page element.

How do you manipulate on element atribues and styles using the DOM?

To manipulate on element atribues and styles using the DOM, we can use JavaScript to access and modify both attributes and CSS of HTML element.
1. we can use Methods like : getAttribute(), setAttribute(), and removeAttribute() to interact element attributes.
2. we can directly modify an elements style properly to change its inline CSS styles. like Set Style: element.style.properly = "value", Add class: element.classList.add("className") atc.
